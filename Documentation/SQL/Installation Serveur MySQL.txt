- Aller sur MySQL Installer
	-> Ajouter un produit MySQL Server 8.0
	-> Télécharger et installer le produit ajouté
	-> Mettre un mot de passe (root par défaut)
	-> Ajouter un utilisateur et choisir localhost sur "Host"
	-> Suivre les étapes par défaut

- Aller sur MySQL Workbench
	-> Créer une connection MySQL
	-> Ajoutez les identifiants (nom du host, port, 
	   utilisateur, mot de passe) mis avec le serveur crée avec 
	   MySQL Installer et Mettez un nom pour la connection
	-> Tester la connection 
	-> Créer un schéma de base de donnée
	-> Cliquer sur le schéma de base de donnée ajouté 
	   (après avoir cliquer sur celle-ci, elle obtient 
	    une police en gras)
	-> Ouvrir le fichier dumpSQL.sql qui est dans 
	   le dossier scripts
	-> Executer le programme
- Aller sur IntelliJ
	-> Installer le plugin "Database Tools and SQL" 
	   si il n'est pas présent
	-> Cliquer sur l'onglet Database 
	-> Sélectionnez "Data Source" > "MySQL".
	-> Ajouter les identifiants mis pour 
	   la connection sur MYSQL Workbench
	-> Tester la connection
	-> Enregistrer la configuration
